# PluginInstalled

Donne le nombre de pluginss installer dans votre systeme

- This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework. 

 ## 🎯 Usage
Commandes :
- combien de plugin installer
- il y'a combien de plugin dans ton syeteme

## Multi-room

The `PluginInstalled` plugin is fully multi-room.

## Multi-language

The `PluginInstalled` plugin relies solely on the system's available languages.
